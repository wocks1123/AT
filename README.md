# AT

- 웹캠, 스마트폰 카메라를 CCTV 처럼 활용해 사용할 수 있는 실시간 객체 탐지 서버
- 화면에 사람이 등장할 경우 해당 구간을 동영상으로 저장
- 브라우저에서 객체 검출을 실행할 수 있다.

## 사용 기술

- FastAPI, NUXT.js, Tailwindcss, Pytorch, Tensorflow.js, Kafka
